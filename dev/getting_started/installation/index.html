Compatibility

This package requires Julia 1.2 or later.

Prerequisites

To make use of the full functionality of SpineOpt, we strongly recommend the installation of Spine Toolbox. SpineToolbox provides all the necessary tools for data management required by SpineOpt.

How to Install

julia> using Pkg

julia> pkg"registry add https://github.com/Spine-project/SpineJuliaRegistry"

julia> pkg"add SpineOpt"

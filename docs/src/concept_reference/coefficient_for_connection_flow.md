The [coefficient\_for\_connection\_flow](@ref) is an optional parameter that can be used to include the [connection\_flow](@ref) variable from or to a [node](@ref) in a [user\_constraint](@ref) via the
[connection\_\_from\_node\_\_user\_constraint](@ref) and [connection\_\_to\_node\_\_user\_constraint](@ref) relationships.
Essentially, [coefficient\_for\_connection\_flow](@ref) appears as a coefficient for the
[connection_flow](@ref) variable from or to the [node](@ref) in the [user constraint](@ref constraint_user_constraint).

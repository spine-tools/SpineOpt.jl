Triggers [flow ratio constraint](@ref constraint_ratio_unit_flow) generation
of the form `flow1 == ratio * flow2`.
Additional terms can be included through
[flow\_ratio\_equality\_online\_coefficient](@ref) for [units\_on](@ref var_units_on) variables,
and [flow\_ratio\_start\_flow](@ref) for [units\_started\_up](@ref var_units_started_up) variables.

See also [flow\_ratio\_greater\_than\_coefficient](@ref),
[flow\_ratio\_less\_than\_coefficient](@ref),
and [How to define an efficiency](@ref).

[unit\_flow\_\_user\_constraint](@ref) is a relationship between a [unit\_flow](@ref) and a [user\_constraint](@ref). The relationship specifies that the [unit\_flow variable](@ref var_unit_flow) between the specified [unit](@ref) and the specified [node](@ref) is involved in the specified [user\_constraint](@ref). Parameters on this relationship generally apply to this specific [unit\_flow variable](@ref var_unit_flow). For example the parameter [coefficient\_for\_unit\_flow](@ref) defined on [unit\_flow\_\_user\_constraint](@ref) represents the coefficient on the specific [unit\_flow variable](@ref var_unit_flow) in the specified [user_constraint](@ref)

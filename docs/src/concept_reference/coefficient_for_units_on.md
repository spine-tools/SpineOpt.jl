The [coefficient\_for\_units\_on](@ref) is an optional parameter that can be used to include the [units\_on](@ref) variable of a [unit](@ref) in a [user\_constraint](@ref) via the [unit\_\_user\_constraint](@ref) relationship.
Essentially, [coefficient\_for\_units\_on](@ref) appears as a coefficient for the
[units\_on](@ref) variable of the [unit](@ref) in the [user constraint](@ref constraint_user_constraint).

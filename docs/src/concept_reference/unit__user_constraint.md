[unit\_\_user\_constraint](@ref) is a two-dimensional relationship between a [unit](@ref) and a [user\_constraint](@ref).
The relationship specifies that a variable or variable(s) associated only with the unit (not a [unit\_flow](@ref var_unit_flow) for example) are involved in the constraint.
For example, the [coefficient\_for\_units\_on](@ref) defined on [unit\_\_user\_constraint](@ref) specifies the coefficient of the [unit](@ref)'s [units\_on](@ref var_units_on) variable in the specified [user\_constraint](@ref).

See also [user\_constraint](@ref).

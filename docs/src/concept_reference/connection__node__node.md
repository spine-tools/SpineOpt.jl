[connection\_\_node\_\_node](@ref) is a three-dimensional relationship between a [connection](@ref), a [node](@ref) (node 1) and another [node](@ref) (node 2). [connection\_\_node\_\_node](@ref) infers a conversion and a direction with respect to that conversion. Node 1 is assumed to be the input node and node 2 is assumed to be the output node. For example, the [fix\_ratio\_out\_in\_connection\_flow](@ref) parameter defined on [connection\_\_node\_\_node](@ref) relates the output [connection\_flow](@ref var_connection_flow) to node 2 to the input [connection\_flow](@ref var_connection_flow) from node 1

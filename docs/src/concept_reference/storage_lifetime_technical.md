Duration parameter that determines the minimum duration of storage investment decisions. Once a storage has been invested-in, it must remain invested-in for [storage\_lifetime\_technical](@ref). Note that [storage\_lifetime\_technical](@ref) is a dynamic parameter that will impact the amount of solution history that must remain available to the optimisation in each step - this may impact performance.

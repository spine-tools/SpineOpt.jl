[node\_\_user\_constraint](@ref) is a two-dimensional relationship between a [node](@ref) and a [user\_constraint](@ref).
The relationship specifies that a variable associated only with the node (currently only the [node\_state](@ref var_node_state)) is involved in the constraint.
For example, the [coefficient\_for\_node\_state](@ref) defined on [node\_\_user\_constraint](@ref) specifies the coefficient of the [node](@ref)'s [node\_state](@ref var_node_state) variable in the specified [user\_constraint](@ref).

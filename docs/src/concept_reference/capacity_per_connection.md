Defines the upper bound on the corresponding [connection\_flow](@ref var_connection_flow) variable. If the connection is a candidate connection, the effective [connection\_flow](@ref var_connection_flow) upper bound is the product of the investment variable, [connections\_invested\_available](@ref var_connections_invested_available) and [capacity\_per\_connection](@ref). If PTDF based DC load flow is enabled, [capacity\_per\_connection](@ref) represents the normal rating of a [connection](@ref) (line) while [connection\_emergency\_capacity](@ref) represents the maximum post contingency flow.

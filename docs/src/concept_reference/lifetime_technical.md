Connection: Duration parameter that determines the minimum duration of [connection](@ref) investment decisions. Once a [connection](@ref) has been invested-in, it must remain invested-in for [lifetime\_technical](@ref).

Unit: Duration parameter that determines the minimum duration of [unit](@ref) investment decisions. Once a [unit](@ref) has been invested-in, it must remain invested-in for [lifetime\_technical](@ref).

Note that [lifetime\_technical](@ref) is a dynamic parameter that will impact the amount of solution history that must remain available to the optimisation in each step - this may impact performance.

See also [Investment Optimization](@ref) and [investment\_count\_max\_cumulative](@ref)
